# Data Dictionary

The data is synthetic and shaped around partnership-marketing integration workflows.

| Field | Meaning |
|---|---|
| `partner_id` | Synthetic affiliate, creator, advocate, or publisher partner identifier |
| `integration_type` | API, CSV, webhook, or pixel feed |
| `event_count` | Daily tracked conversion or click events |
| `duplicate_rate` | Share of events suspected to be duplicates |
| `schema_drift_count` | Count of unexpected or missing payload fields |
| `payout_gap` | Difference between tracked commission and expected payout |
| `severity_score` | Weighted reliability-risk score from 0 to 100 |
| `customer_impact` | Estimated affected advertiser or partner accounts |
